# Práctica 3 - Sistemas Operativos 2026A

**Integrantes:** Karely Bombón, Jhon Tiupul, Fidel Quilumba, Jhoan Sasnalema

**Profesor:** Ing. Diana Martínez, PhD.

**Tema:** Creación de Procesos

## Objetivos:
1. Familiarizar al estudiante con el uso de las funciones fork, system, exec.
2. Realizar varias actividades de creación de procesos padres e hijos.
3. Comprender la jerarquía de procesos, el uso de identificadores (PID/PPID) y los estados de los procesos (huérfanos y zombies).

## Contenido
1. **Uso de Fork:** Implementación de creación de procesos, manejo de variables compartidas y gestión de estados (huérfanos/zombies).
2. **Uso de Exec:** Uso de la familia de funciones `exec` para sustituir la imagen de un proceso.
3. **Uso de System:** Ejecución de comandos del sistema operativo desde código en C.
